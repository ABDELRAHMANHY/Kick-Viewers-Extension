# Kick Viewers & Chat Extension 🟢

إضافة متصفح خفيفة وآمنة لمنصة Kick.com، تتيح لك متابعة عدد المشاهدين الحقيقيين وعداد الرسائل الحية مباشرة من واجهة البث دون الحاجة لفتح أدوات معقدة.

## ✨ المميزات 
* **عرض المشاهدين الحقيقيين:** يعتمد على الـ API الرسمي لمنصة كيك.
* **عداد الشات الحي:** متابعة نشاط الدردشة في الوقت الفعلي.
* **واجهة بسيطة:** تشغيل وإيقاف بضغطة زر.
* **أداء فائق:** لا يستهلك موارد المتصفح (RAM/CPU).

## 🚀 طريقة التثبيت
بما أن الإضافة في نسختها المستقلة، يمكنك تثبيتها بسهولة:

1. حمل ملف الإضافة بصيغة ZIP من قسم الإصدارات (Releases) على اليمين.
2. قم بفك الضغط عن الملف في مجلد على جهازك.
3. افتح متصفح كروم واذهب إلى الرابط: `chrome://extensions/`
4. فعّل **وضع المطورين (Developer mode)** في الزاوية العلوية.
5. اضغط على **Load unpacked** واختر المجلد الذي قمت بفك الضغط عنه.
6. الإضافة جاهزة للعمل!
